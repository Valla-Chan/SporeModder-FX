SporeModder FX

The most advanced and modern Spore data modding tool.

Open, browse, and edit the contents of Maxis Database Packed Files (DBPFs), also known as .package files.

Enhanced editing support for a wide variety of custom Maxis file formats used in Spore.

Requires Java 11 or newer

User-friendly

Simple, easy and pretty. A modern user interface for a modern tool.

Easy coding

A new format that makes coding .prop files easier than ever. Includes error highlighting, so you can instantly detect what you are doing wrong. And autocomplete, to make coding a bit faster and more convienient.
